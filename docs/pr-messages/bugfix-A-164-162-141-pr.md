## 📝 Description

### 🔄 What Changed
- **`src/shared/type-guards.js`** (#164): Added `!Array.isArray(value)` guard to `isRecord()` to reject arrays
- **`tests/unit/shared/type-guards.test.js`**: Updated test to expect `isRecord([]) === false`
- **`package.json`** (#162): Set `"private": true` to prevent accidental npm publish
- **`package.json`** (#141): Removed duplicate `"coverage"` script (alias for `"test:coverage"`)
- **`tests/e2e/audit/audit.e2e.test.js`**: Added regression tests for #162 and #141

### 🎯 Why
- **SEC-05 (#164)**: `isRecord()` returned `true` for arrays (`typeof [] === 'object'`), eroding defense-in-depth for map/asset validation
- **SEC-03 (#162)**: `"private": false` allowed accidental `npm publish` of GPL-3.0 code and internal config
- **DEAD-05 (#141)**: Duplicate `"coverage"` script alias wasted maintainer attention
